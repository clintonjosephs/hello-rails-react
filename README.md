# hello-react-rails

## Project Description

This project contains the first trial at connecting React on the front-end to Ruby-on-Rails on the backend. The app displays random messages contained in the database table on a click of a button.


## Built With

- Ruby On Rails
- React-Rails gem
- Rubocop
- Git
